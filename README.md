# 主要功能    
## 1.自動掛機您想要掛的地點  
## 2.自動回傳經驗值至DC  
## 3.自動宣傳訊息(設施宣傳&交易頻道)  
## 4.可以不用開遊戲輸入指令&聊天  
## 5.可以透過私訊Bot來執行命令(須設定白名單)  
## 6.新增白名單指令系統(避免有心人士亂搞)  
## 7.許多自定義指令   
## 8.自動接受tpa & tpahere(需白名單)  
## 10.丟棄物品(可指定和全部丟棄)  
## 11.自動重連(防止廢土掛掉的時候無法重連)
